# ATM Machine  

A simple **ATM simulation project in Java** that allows users to check their balance, withdraw money, and view transaction details.  

---

## 🚀 Features  
- 💰 View available balance  
- ➖ Withdraw money  
- 📜 Transaction details  
- 🔒 Exit option  

---

## 🛠️ Tech Stack  
- **Java** (Core Java, OOP concepts)
